# Gestor de Tareas en Python (CLI)

¡Hola! Este es mi primer proyecto de programación independiente mientras curso la Licenciatura en Informática. Es una aplicación de consola (CLI) diseñada para gestionar tareas diarias, ideal para ejecutar en entornos móviles como **Pydroid 3** o desde la terminal de la computadora.

El objetivo de este proyecto fue aplicar los conceptos fundamentales de la algorítmica y las estructuras de datos dinámicas en un lenguaje del mundo real como Python.

## 🚀 Características
* **Crear:** Permite añadir nuevas tareas a la lista en tiempo real.
* **Leer:** Muestra la lista completa de tareas pendientes con una numeración dinámica.
* **Borrar:** Elimina tareas específicas seleccionando su número de índice.
* **Control de Flujo:** Menú interactivo basado en un bucle infinito controlado por el usuario.

## 🛠️ Conceptos Algorítmicos Aplicados
* **Estructuras de Datos:** Uso de listas dinámicas (`[]`) para el almacenamiento en memoria.
* **Estructuras de Control:** Bucles condicionales (`while True`) y bifurcaciones lógicas (`if / elif / else`).
* **Manejo de Tipos de Datos:** Conversión de tipos de datos de texto (String) a enteros (Integer) para el manejo de índices.
* **Funciones Integradas:** Uso de métodos nativos de Python como `.append()`, `.pop()` y `enumerate()`.
